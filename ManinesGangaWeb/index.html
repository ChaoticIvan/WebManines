PRESENTACIÓN


Los Manines Ganga es un joven equipo artístico multidisciplinar que se compromete con el futuro de sus proyectos y empresas como si este fuera suyo propio.


VALORES

Originalidad

Creemos que la manera de hacer destacar un proyecto parte desde su propia concepción; evaluar todas las ideas buscando la más destacable y original.

Compromiso

Nuestro equipo toma cada proyecto como si fuera suyo propio: cuando se fija un objetivo es el mismo para todos, y haremos lo posible para conseguirlo.

Puntualidad

Es muy importante para nosotros el definir las etapas de un proyecto; nos enorgullecemos de decir que nos organizamos las entregas para el plazo previsto y empezamos desde el primer momento a trabajar.

Calidad

Somos perfeccionistas: el proyecto no se acaba cuando se cumplen los requisitos sino cuando estamos totalmente conformes con el trabajo realizado.
